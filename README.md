# Super Search — 跨平台超级搜索

Reasonix 技能：一键搜索 B站·知乎·微博·GitHub·小红书·全网，对比分析，深度研究。

## 功能

- 🔍 **6 平台并行搜索**：B站/知乎/微博/GitHub/小红书/全网
- 📊 **跨平台对比**：结构化表格，发现观点分歧
- 🔬 **深度研究**：三阶段递归搜索 → 自动生成报告
- 👀 **话题追踪**：关注关键词，持续监控变化
- 📜 **搜索历史**：自动保存每次搜索
- 📄 **报告导出**：一键导出 Markdown 研究到桌面
- ✅ **可信度标注**：多源验证确认/单源/矛盾

## 使用

| 你说 | 效果 |
|------|------|
| 「搜一下AI绘画」 | 快速全网搜索 |
| 「各平台怎么说iPhone17」 | 跨平台对比表 |
| 「深度研究量子计算」 | 三阶段研究+报告 |
| 「关注英伟达股价」 | 保存话题，持续追踪 |
| 「我的搜索」 | 查看搜索历史 |
