# Super Search v3.0 — Agent-Reach 驱动版

B站·小红书·Twitter·Reddit·GitHub·YouTube·全网 多平台并行搜索。

> ⚠️ 需要先安装 [Agent-Reach](https://github.com/Panniantong/agent-reach)

## 安装（两步）

```
第1步：安装 Agent-Reach（搜索引擎）
/install-capability https://raw.githubusercontent.com/Panniantong/agent-reach/main/docs/install.md

第2步：安装 Super Search（搜索技能）
/install-capability https://github.com/uyiuhkjuhui/super-search
```

## 功能

bili · opencli · curl/Jina · gh · yt-dlp — 原生 API 驱动，中文不乱码，全平台大量搜索。

## 使用

| 说 | 效果 |
|----|------|
| 搜一下辽宁工程技术大学 | 全平台并行，20条+结果 |
| 各平台怎么说iPhone17 | 跨平台对比表 |
| 深度研究量子计算 | 递归搜索+自动报告 |
